# Product Overview

New Avalon: Skirmish is a virtual tabletop application for a multiplayer card game. It provides a digital implementation of a tactical card game where players deploy units, use abilities, and compete to score points by forming lines on a grid-based board.

## Core Features

- **Multiplayer gameplay**: Real-time WebSocket-based multiplayer supporting up to 4 players
- **Game modes**: Free-for-all, 2v2 teams, and 3v1 asymmetric modes
- **Deck system**: Multiple playable factions (SynchroTech, Hoods, Optimates, Fusion) with unique cards
- **Turn-based phases**: Setup, Main, and Commit phases with scoring mechanics
- **Card abilities**: Deploy abilities, persistent effects, and command cards with complex targeting
- **Status system**: Counters and status effects (Aim, Stun, Power modifiers, etc.)
- **Spectator mode**: Players can watch ongoing games
- **Custom decks**: Support for custom deck building and loading
- **Reconnection**: Players can disconnect and rejoin games without losing progress
- **Localization**: Multi-language support (currently English and Russian)

## Target Use Case

Designed for remote play sessions where players want to experience the physical card game digitally, with full rules enforcement and visual feedback for game state changes.
