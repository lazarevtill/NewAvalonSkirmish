# New Avalon: Skirmish

A virtual tabletop application for the New Avalon: Skirmish multiplayer card game. Play tactical card battles with friends in real-time over WebSocket connections.

## Features

- Real-time multiplayer for up to 4 players
- Multiple game modes (Free-for-all, 2v2, 3v1)
- Multiple playable factions with unique cards
- Spectator mode
- Custom deck building
- Player reconnection support
- Multi-language support (English, Russian)

## Quick Start

**Prerequisites:** Node.js 18+

### Development Mode

```bash
npm install
npm run dev
```

Opens at: http://localhost:5173

### Production Mode

```bash
npm install
npm run build
npm start
```

Opens at: http://localhost:8080
